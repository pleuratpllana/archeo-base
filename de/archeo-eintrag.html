Archeo Eintrag

Anwendung zur Erfassung archäologischer Informationen

ArcheoEntry ist das einzige Tool zur Eingabe von Informationen, das sich vollständig an Ihre Bedürfnisse anpasst. Seine Oberfläche reduziert die Zeit, die für die Verwaltung von Felddaten und die Analyse nach der Ausgrabung benötigt wird, erheblich.

Formulare, die auf Ihre Bedürfnisse zugeschnitten sind.

Erstellen Sie Formulare selbst oder wählen Sie Vorlagen aus einem gemeinsamen Register.

Dynamische Formulare

Die Formulare sind leicht vom Benutzer konfigurierbar und können problemlos an die individuellen Bedürfnisse jeder Organisation und jeder Ausgrabungsstätte angepasst werden. Der Komplexität der Formulare, die Sie erstellen können, sind keine Grenzen gesetzt. Drag-and-Drop-Menüs, einfache Felder, Text, Zahlen, Datum, Skizzen, Register, Tabellen ...

Konfigurierbare Tabelle

Die Formulardaten können in Tabellenform mit einer benutzerdefinierten Auswahl an Spalten angezeigt werden. Auf dieser Basis können die Daten dann einfach in verschiedene Formate (xls, csv...) exportiert werden.

Formularregister

Die erstellten Formulare können intern gespeichert oder über ein Formularregister mit anderen Nutzern oder Institutionen geteilt werden.

Arbeiten von überall aus

Sofortige Synchronisierung zwischen den Nutzern an der Ausgrabungsstätte und im Büro.

Zentrale Verwaltung

Die Webanwendung ermöglicht die Live-Synchronisation von Daten, unabhängig von der Anzahl der Nutzer an der gleichen Ausgrabungsstelle. Beobachten Sie den Fortschritt der Ausgrabung und arbeiten Sie von Ihrem Büro aus.

Webbasierte Schnittstelle

ArcheoBase kann von jedem Webbrowser aus genutzt werden. Der Benutzer hat die gleiche Arbeitsumgebung, ob im Büro oder mit einem Tablet an der Ausgrabungsstelle.

Einmalige Eingabe

Eintragungen auf Papier sind nicht mehr erforderlich. Die einmalige Eingabe von Daten spart eine Menge Zeit.

Gemeinsame Nutzung von Daten

ArcheoBase erleichtert den Austausch von Informationen zwischen Institutionen. Der Nutzer kann festlegen, ob er seine Daten innerhalb seiner eigenen Institution halten oder sie anderen Nutzern oder Institutionen zugänglich machen möchte.

Gruppenprojekte

Nutzer aus verschiedenen Institutionen oder Einzelpersonen können ein Gruppenprojekt beginnen und zusammenarbeiten.

Integriertes GIS

Visualisieren Sie Ihre georeferenzierten Daten und lassen Sie sich Ihre Orthofotos anzeigen.

Überblick und Visualisierung auf der Karte

Die eingegebenen Daten werden, sofern sie georeferenziert sind, automatisch in ein geografisches Informationssystem (GIS) integriert.

Sig-Import

Der Import von Topospunkten und deren Anzeige auf der Karte kann mithilfe eines Import-Tools aus XLSX- oder CSV-Dateien durchgeführt werden.

Orthophotos

Die hochgeladenen Orthofotos werden automatisch gefiltert. So können Sie sie sich im Webmodus bei gleichbleibend hoher Qualität anschauen. Die Originale können vom Nutzer jederzeit heruntergeladen werden.

Sicherheit

Datenschutz, Speicherung, Verfügbarkeit, Rückverfolgbarkeit, Backup und Sicherheit auf allen Ebenen.

Belastbarkeit

Das Programm wurde so konzipiert, dass es große Datenlasten bewältigen kann und dabei eine hervorragende Ausführungsgeschwindigkeit beibehält. Die Speicherung von Daten, Dokumenten und Fotos ist unbegrenzt.

Nachvollziehbarkeit

Keine Information geht verloren. Die Rückverfolgbarkeit aller Änderungen wird durch ein automatisches Backup-System gewährleistet, das es ermöglicht, in der Zeit zurückzugehen, um die vorgenommenen Änderungen zu identifizieren.

Rechenleistung

Die Rechenleistung ist sofort für alle Verfahren verfügbar, die eine solide Grundlage auf dieser Ebene erfordern (Bildverarbeitung, künstliche Intelligenz usw.).

Spezifizierbare Rechte

Mit einem ausgeklügelten Rechtesystem können Sie einfach definieren, welche Benutzer Zugang zu den Informationen einer Ausgrabungsstelle haben und ihnen Rollen (Lesen, Schreiben, Admin) zuweisen.

Stockage redondant

Le stockage des données est redondant et hautement sécurisé avec des sauvegardes quotidiennes.

Datensicherungsmodul

Auf Nachfrage kann ein automatisiertes Backup- und Exportmodul zur Verfügung gestellt werden. Dieses ermöglicht den regelmäßigen Export aller in ArcheoBase enthaltenen Informationen auf einen anderen Server.

Andere Funktionen

Skizzen

Um das Erstellen von Skizzen zu erleichtern, kann der Benutzer mit der Hand auf ein Foto im Hintergrund zeichnen. Ein solches Foto kann direkt mit dem Tablet aufgenommen oder aus dem Fotoverzeichnis abgerufen werden. Bilder von Skeletten wurden vorverarbeitet, sodass der Nutzer die gefundenen Überreste leicht markieren kann.

Spracheingabe

Die Schnelligkeit und Einheitlichkeit der Eingabe wird durch Dropdown-Menüs gewährleistet. Dieses System wird durch ein Spracheingabetool ergänzt, das die vom Benutzer diktierten Elemente direkt in die Eingabefelder überträgt.

Gemeinsame Nutzung von Tablets

Um die Kosten für die Hardware zu begrenzen, können sich die Nutzer vor Ort ein Tablet teilen. Ein schnelles Log-in-System ermöglicht einen schnellen Benutzerwechsel.

Bilderkennung

Ein System zur Erkennung von Zeichen auf einem Bild ermöglicht es, Fotos, die mit einer Referenzplakette aufgenommen wurden, automatisch mit den entsprechenden archäologischen oder technischen Einheiten zu verknüpfen.

Mehrsprachigkeit

Wie alle anderen ArcheoBase-Modelle ist auch ArcheoEntry in verschiedenen Sprachen verfügbar (Deutsch, Französisch, Englisch, Italienisch).

Ein Blick in die Zukunft

Kontinuierliche Entwicklung

Es werden laufend neue Funktionen implementiert. Die Prioritäten der Entwicklung können auf der Seite „Mein Konto“ eingesehen werden.

Gemeinschaftlicher Ansatz

ArcheoBase lebt von einem partizipativen Ansatz, der die Vorschläge der Nutzer einbezieht. Technische Lösungen, die den identifizierten Bedürfnissen entsprechen, werden direkt implementiert und der gesamten Nutzergemeinschaft zur Verfügung gestellt.
