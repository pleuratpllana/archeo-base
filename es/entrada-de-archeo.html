Entrada de Archeo

Aplicación de introducción de información arqueológica

ArcheoEntry es la única herramienta de introducción de información que se adapta completamente a tus necesidades. Su dinámica reduce en gran medida el tiempo necesario para la gestión de datos de excavación y el análisis posterior de ésta. Se ha diseñado sobre bases tecnológicas muy sólidas para poder gestionar grandes sitios y el ingreso simultáneo de un gran número de usuarios.

Formularios adaptados a Tus Necesidades

Crea tus propios formularios o elige uno de los del registro compartido

Formularios dinámicos

El usuario puede configurar fácilmente los formularios y se pueden adaptar fácilmente a las necesidades individuales de cada organización y cada sitio. No hay límites en cuanto a la complejidad de los formularios que se pueden crear. Arrastra-coloca menús desplegables, campos simples, texto, numérico, fecha, croquis, pestaña, tabla...

Tabla configurable

Los datos del formulario se pueden ver en tablas, con una selección de columnas definidas por el usuario. Sobre esta base, los datos se pueden exportar fácilmente en diferentes formatos (xls, csv...).

Registro de formularios

Los formularios creados se pueden almacenar de forma privada o compartir con otros usuarios o instituciones a través de un registro de formularios.

Trabaja desde cualquier lugar

Sincronización instantánea entre los usuarios de la excavación y de la oficina

Gestión centralizada

La aplicación web permite la sincronización de datos en vivo, independientemente de la cantidad de usuarios en el mismo sitio. Observa el progreso del sitio de excavación y trabaja desde tu oficina.

Interfaz web

ArcheoBase se puede utilizar desde cualquier navegador web. El usuario se beneficia del mismo entorno de trabajo, ya sea en la oficina o con una tableta en el terreno.

Introducción única

La introducción por escrito en papel ya no es necesaria. Introducir los datos una sola vez ahorra un tiempo considerable.

Compartir datos

ArcheoBase facilita el intercambio de información entre instituciones. El usuario puede definir si quiere mantener sus datos dentro de su propia institución o si quiere hacerlos accesibles a otros usuarios o instituciones.

Proyectos de grupo

Los usuarios de diferentes instituciones o individuos pueden crear un proyecto de grupo y trabajar juntos.

SIG integrado

Visualiza tus datos georreferenciados, visualiza tus ortofotos

Resumen y presentación en el mapa

Los datos ingresados, si están georreferenciados, se integran automáticamente a un sistema de información geográfica (SIG).

Importación SIG

La importación de puntos topográficos y su visualización en el mapa se puede realizar mediante un procedimiento de importación desde archivos XLSX o CSV.

Ortofotos

El sistema ordena las ortofotos cargadas para mostrarlas en el mapa en modo Web, manteniendo un nivel de calidad muy alto. Los originales siguen siendo descargables en cualquier momento por el usuario.

Seguridad

Derechos, almacenamiento, disponibilidad, rastreabilidad, copia, seguridad a todos los niveles

Capacidad de carga

El programa ha sido diseñado para poder soportar grandes cargas de datos, manteniendo una excelente velocidad de ejecución. El almacenamiento de datos, documentos y fotos es ilimitado.

Rastreabilidad

No se pierde ninguna información. La rastreabilidad de todas las modificaciones está garantizada por un sistema de copia de seguridad automático que permite retroceder en el tiempo para identificar los cambios realizados.

Potencia de procesamiento

La potencia de procesamiento está disponible instantáneamente para todos los procedimientos que requieren bases sólidas a este nivel (procesamiento de imágenes, inteligencia artificial, etc.).

Derechos especificables

Un sofisticado sistema de derechos permite definir los usuarios que tienen acceso a la información de un sitio así como las acciones (leer-escribir-administrar) que se les otorga

Almacenamiento redundante

El almacenamiento de datos es redundante y muy seguro con copias de seguridad diarias.

Copia de seguridad

Para las instituciones que lo deseen, podrán disponer de un módulo automatizado de copia de seguridad y exportación. Lo que permitirá exportar periódicamente toda la información contenida en ArcheoBase a otro servidor.

Otras funcionalidades

Croquis

Para facilitar el croquis, el usuario puede dibujar a mano sobre una foto en segundo plano. Dicha foto se puede tomar directamente con la tableta o escoger del directorio de fotos. Las imágenes de los esqueletos se han preprocesado para que el usuario pueda resaltar fácilmente los huesos descubiertos.

Entrada de voz

La velocidad y uniformidad de la entrada se garantiza a través de menús desplegables. Este sistema se complementa con una herramienta de entrada de voz que transcribe directamente los elementos dictados por el usuario en los campos de entrada.

Compartir tableta

Para limitar los gastos de los materiales, los usuarios pueden compartir la misma tableta en el terreno. El sistema de inicio de sesión rápido permite cambiar fácilmente de usuario.

Reconocimiento de imagen

Sistema de detección de caracteres en una imagen que permite vincular automáticamente las fotos que se han tomado con una placa de referencia y las unidades arqueológicas o técnicas relacionadas.

Multilingüismo

Como todos los demás módulos de ArcheoBase, ArcheoEntry está disponible en diferentes idiomas (francés, alemán, inglés, italiano).

Una mirada hacia el futuro

Desarrollo continuo

Las nuevas funcionalidades se implementan continuamente. Se han establecido prioridades en términos de desarrollo y se pueden ver en la página "Mi cuenta".

Enfoque comunitario

ArcheoBase se mantiene a través de un enfoque participativo al integrar las sugerencias de los usuarios. Las soluciones técnicas que responden a las necesidades identificadas se implementan directamente y se ponen a disposición de toda la comunidad de usuarios.
